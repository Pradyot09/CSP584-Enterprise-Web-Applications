Step 1: Download and install tomcat apache server as app server by by fowlling the guidelines as discussed in the class.

Step 2: Download and install MySQL on your local system by fowlling the guidelines as discussed in the class.

Step 3: Create a database name smartportables.

Step 4: In database smartportables create three tables Registration, ProductDetails, CustomerOrders in MySQL using "Create Table" query with required fields,for userdetails, products and orders.

Step 5: Download and install MongoDB database server on local machine by fowlling the guidelines as discussed in the class to store review of the products.

Step 6: Create a database using command "Use databasename" eg: use CustomerReview.

Step 7: Create a collection under database CustomerReview using the command "db.createCollection("collectionname")" eg: db.createCollection("myReviews").

Step 8: Copy and paste the folder Homework2 in C:\Tomcat\apache-tomcat-7.0.34\webapps.

Step 9: Start the tomcat Apache server and browse the folder name in the URL followed by localhost. eg: localhost\Homework2.

Step 10: Appplication is live now and you will be able to use all the functionalities discussed for Homewor2.
