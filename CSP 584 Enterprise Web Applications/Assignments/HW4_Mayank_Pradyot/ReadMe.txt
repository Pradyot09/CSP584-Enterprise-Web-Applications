Step 1: Download and install tomcat apache server as app server by by fowlling the guidelines as discussed in the class.

Step 2: Download and install MySQL on your local system by fowlling the guidelines as discussed in the class.

Step 3: Create a database name smartportables.

Step 4: In database smartportables create three tables Registration, Productlist, CustomerOrders in MySQL using "Create Table" query with required fields,for userdetails, products and orders.

Step 5: Copy and paste the folder Homework4 in C:\Tomcat\apache-tomcat-7.0.34\webapps.

Step 6: Start the tomcat Apache server and browse the folder name in the URL followed by localhost. eg: localhost\Homework4.

Step 7: Appplication is live now and you will be able to use all the functionalities discussed for Homework4. 

Step 8: Login as Manager with user name and password.

Step 9:Click on the Add, Delete, Update Products to perform the operation on the table Productlist.
Step 10: Search any products with its name to in the search box and then click on it for further information.
